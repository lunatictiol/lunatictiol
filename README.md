## What I'm building

Stack: LangGraph · FastAPI · n8n · Ollama · Docker Compose.

**[SQL LLM Agent](https://github.com/sabiq-aijaz/sql-llm-agent)** — Schema-grounded natural language to PostgreSQL pipeline. Uses LangChain for SQL generation and DSPy + Phi-3 for deterministic validation — intentionally separating creative generation from structured judgment. Stack: LangChain · DSPy · FastAPI · Streamlit · PostgreSQL.
